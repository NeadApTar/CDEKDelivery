# Настройка

Доступ к настройкам можно получить по ссылке на странице со списком плагинов.

<Image img={require('./img/setup.png')} />

Прежде всего, необходимо заполнить поля `Идентификатор`, `Секретный ключ` и `Ключ Яндекс.Карты` и нажать на кнопку
`Сохранить изменения` для применения изменений.

<Image img={require('./img/settings.png')} />

<details>
    <summary>Где взять `Идентификатор` и `Секретный ключ`</summary>

    Необходимо перейти на страницу [Интеграции](https://lk.cdek.ru/integration) в ЛК СДЭК
    <Image img={require('./img/int.png')} />
    И скопировать ключи из соответствующих полей.
    <Image img={require('./img/keys.png')} />


</details>

<details>
    <summary>Где получить `Ключ Яндекс.Карты`</summary>

    Зайдите на страницу Кабинета Разработчика и нажмите кнопку Получить ключ. Во всплывающем окне выберите сервис
    «JavaScript API и HTTP Геокодер».

    :::note

    Может пройти до 1 часа после последних изменений до того, как ключ начнет работать.

    :::

    После заполнения формы появится надпись «Сервис успешно подключен». Созданный ключ будет доступен в разделе «Ключи».
    Его необходимо использовать в дальнейшем при подключении API.

    :::warning

    Обязательно задайте параметр HTTP Referrer равный адресу вашего сайта для ключа.

    Подробнее можно прочитать на странице https://yandex.ru/dev/jsapi30/doc/ru/limit

    :::
</details>

Необходимо обязательно ввести адрес для отправки груза (заказов) - для тарифов от склада\двери.

Если Вы хотите, чтобы после выбора ПВЗ покупателем карта автоматически закрывалась - активируйте чекбокс "Закрыть карту после выбора пвз".

:::

<Image img={require('./img/address.png')} />

:::tip

Выбор нескольких тарифов происходит через нажатую кнопку CTRL

:::

<details>
    <summary>Карта на сайте не загружается</summary>

    <Image img={require('./img/yandex.png')} />

    Пожалуйста, проверьте ключ карты Яндекс и настройки HTTP Referrer. 

Для этого необходимо:

1.Перейти в настройки я.ключа по ссылке: https://developer.tech.yandex.ru/
2.Навести на ключ и нажать кнопку "Изменить". Скриншот: https://i.imgur.com/8Eu6k2I.png
3.В поле "Ограничение по HTTP Referer" прописать Ваш сайт без указания "https", то есть сайт должен быть прописан таким образом. Скриншот примера: https://i.imgur.com/AFvNbpi.png
4. После сохранения настроек в течении 60 минут карта в модуле отобразится.
Если проблема сохраняется, пожалуйста, обратитесь в [техподдержку](mailto:integrator@cdek.ru).

</details>

В блоке "Услуги" Вы можете подключить к заказам необходимые дополнительные услуги.

В блоке "Cтоимость доставки" установить наценку или скидку на стоимость доставки, добавить к расчету стоимость страховки (рассчитывается как 0,75% от стоимости товаров в заказе).

:::tip

Напротив некоторых полей есть значок вопроса - при наведении на него, отобразится подробная информация о поле

:::
