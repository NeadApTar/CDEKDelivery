# Яндекс.Карты

Для получения ключа, зайдите на страницу Кабинета Разработчика и выберите сервис `JavaScript API и HTTP Геокодер`.

<Image img={require('./img/yandex/select.png')} />

<details>
    <summary>Сервис `JavaScript API и HTTP Геокодер` не отображается в списке</summary>

    Если сервис не отображается в списке, то вам необходимо воспользоваться функционалом его подключения.

    <Image img={require('./img/yandex/new.png')} />

    Затем в списке выбрать сервис и подтвердить подключение.

    <Image img={require('./img/yandex/connect.png')} />

    Заполнив форму, подтвердить ее отправку.

    <Image img={require('./img/yandex/form.png')} />

</details>

Необходимо отредактировать параметры ключа, зайдя в его настройки.

<Image img={require('./img/yandex/edit.png')} />

<details>
    <summary>На странице нет доступных ключей</summary>

    Если доступных ключей нет, вы можете создать новый, воспользовавшись соответствующим функционалом.

    <Image img={require('./img/yandex/create.png')} />

    Заполнить название ключа и подтвердить добавление ключа.

    <Image img={require('./img/yandex/key.png')} />

</details>

В поле `Ограничение HTTP Referrer` необходимо указать домен вашего сайта и сохранить изменения.

:::note

Подробнее про ограничения можно прочитать на странице https://yandex.ru/dev/jsapi30/doc/ru/limit

:::

<Image img={require('./img/yandex/save.png')} />

:::warning

Может пройти до 1 часа после последних изменений до того, как ключ начнет работать.

:::

Полученную строку ключа необходимо использовать в настройках плагина.

<Image img={require('./img/yandex/export.png')} />
