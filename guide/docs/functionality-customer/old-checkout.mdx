# Страница оформления заказа на основе шорткода

На этой странице описывается, как плагин работает с классической страницей оформления заказа.

<details>
    <summary>Как понять какую версию страницы оформления заказа я использую?</summary>

    Вы можете проверить тип страницы оформления заказа в панели администратора.
    Смотрите наш [гайд](./checkout-versions) для более подробной информации.

</details>

После добавления товаров в корзину, на странице оформления заказа вы найдете варианты доставки с различными тарифами.
При изменении адреса доставки, цена доставки будет пересчитана.

<Image img={require('./img/old.png')} />

Для тарифов "на склад" и "на пункт выдачи" появится кнопка **Выбрать ПВЗ**, при клике на которую появится карта с
возможностью выбрать конкретную точку получения.

<Image img={require('./img/button.png')} />

:::warning

В зависимости от используемой темы, позиция блоков на странице оформления заказа может отличаться.
Однако, в целом, кнопка карты должна находиться рядом с вариантами доставки.

:::

Выбранный ПВЗ будет отображен рядом с кнопкой **Повторно выбрать ПВЗ**

<Image img={require('./img/pickup.png')} />

:::warning

Выше описан функционал для классической версии страницы оформления заказа.
Новая версия страницы оформления заказа имеет другое отображение карты.
Пожалуйста, обратитесь к [гайду по новой странице оформления заказа](./new-checkout) для получения более подробной
информации.

:::
