# Cash On Delivery

The plugin supports the cash-on-delivery service.
To set payment methods, go to `WooCommerce`->`Settings`->`Payments`, enable **Cash on delivery**.

<Image img={require('./img/list.png')} />

After enabling, you need to assign CDEK delivery method to the cash-on-delivery payment method.
