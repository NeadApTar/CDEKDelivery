# Introduction

This plugin provides integration with CDEK delivery service for Woocommerce stores

## What you'll need

- [PHP](https://www.php.net/downloads) version 7.4 or above
- [MySQL](https://www.mysql.com/downloads/) version 5.67 or above OR
  - [MariaDB](https://mariadb.org/download/) version 10.4 or above
- [Wordpress](https://wordpress.org/download/) version 6.3 or above
- [Woocommerce](https://wordpress.org/plugins/woocommerce/) version 6.9 or above
- [CDEK account](https://lk.cdek.ru/) with enabled [Integration API](https://lk.cdek.ru/integration)

:::warning

Don't forget to check [Woocommerce Requirements](https://woocommerce.com/document/server-requirements/#section-1) too

:::

## Installing plugin

To install, follow the steps below:

1. In your admin panel go to: `Plugins` -> `Add New Plugin`
1. Search for `CDEKDelivery`
1. Click **Install Now**
1. Wait for **Install Now** button becomes **Activate**
